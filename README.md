About
A Python web app that scrapes websites for job listings, stores them in a database, and shows results and email notifications.




WV Job Scraper - Project Reflection
Why I Built This Project:
As someone currently pursuing a career in software development, I found it difficult to efficiently search for software development, programmer, and internship opportunities in this state because of the large amount of old listings. Job postings were spread across multiple webstes which took a lot of time and effort to make sure no companies were missed. While this project solved that problem for me, it also provided me an opportunity to help those around me. West Virginia is a dry state for computer science careers and the intention of this project is to make it easier for people to connect to employers and vice versa. 
I created this Job Board Scraper to centralize relevant jobs and provide a simpler way for students and job seekers to discover openings. 



Technologies Used:

Challenges and Lessons Learned
This project helped me learn how multiple parts of a web application work together. 
I learned how to:
-Scrape, process and clean information from multiple websites
-Store and retrieve information from a database
-Build a web application using Flask
-Implemented email notification functionality and administration controls for sending job updates
-Manage applications configuration using environment variables
-Use Git and GitHub for version control
-Debug issues related to dynamic websites that required Selenium instead of traditional requests



Future Improvements include:
-Additional job sources
-Improved filtering options
-Enhanced email subscription management
-Automated scheduled refreshes
-Public deployment for broader access
-Option to export as CSV



What this project taught me:
The most valuable lesson I learned from this project was understanding how indicidual technologies interact and fit together to create a complete application. Rather than focusing on a single programming concept, I gained experience working accross databases, web development, and deployment consideration within one project.
